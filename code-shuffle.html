CODE SHUFFLE 2025

A fun yet challenging coding event where logic meets twist! Rearranged code snippets, scrambled logic, and tricky bugs await. Decode, debug, and solve in record time to win.

Register Now

Venue

First Floor, Lab 110, D.Y. Patil School of Engineering & Management, Kolhapur

Rules & Regulations

Eligibility

  • Open to all college students (undergraduate, postgraduate & Diploma).
  • Participation is solo (1 participant).
  • Knowledge of Python programming is required.

Event Structure

  • 1. Aptitude Round:
    • 30 questions
    • 1 mark each
    • 30 minutes
    • Topics: C, Data Structure, Pointers
  • 2. Debugging Round:
    • 60 minutes
    • Code with errors will be provided
    • Participants need to identify and submit the corrected code
    • Language: C
  • 3. Code Shuffle Round:
    • 2 questions in C
    • 3 questions in Python
    • Random code statements will be provided
    • Participants need to arrange the code in the correct order and execute it

Scoring & Rules

  • Fast Finish Bonus: Extra points for completing the coding round early.
  • No cheating: No use of AI or external help during any round.
  • Highest score in the coding puzzle round wins.
  • Tie Breaker: Fastest completion time in the coding round.
  • Judges' decisions are final.
